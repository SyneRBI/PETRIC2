# Siemens_mMR_ACR2

A second dataset from Pawel Markiewicz of the same phantom
as the previous ACR acquisition, without moving the phantom.
The difference is that here an hot source (outside the FOV) was added,
and therefore the randoms background is a lot higher.

Before the scripts, KT copied the reg_mumap.*v previously
supplied by Pawel for the ACR dataset.
